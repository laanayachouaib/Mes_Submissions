# À propos du projet

## Présentation
Ce projet est un exercice d'apprentissage de Git et GitHub. Il contient une série de challenges pratiques pour maîtriser les concepts fondamentaux du version control.

## Objectifs du projet
- Apprendre les bases de Git
- Maîtriser le workflow avec les branches
- Comprendre les bonnes pratiques de versionnement
- Préparer à la collaboration en équipe

## Technologies utilisées
- **Git** : Système de version control
- **GitHub** : Plateforme d'hébergement
- **VS Code** : Éditeur de code
- **Markdown** : Format de documentation
