# Documentation .gitignore

## Objectif
Le fichier \`.gitignore\` permet de spécifier quels fichiers Git doit ignorer et ne pas versionner.

## Règles configurées

### Fichiers temporaires
\`\`\`
*.tmp      # Tous les fichiers .tmp
*.log      # Tous les fichiers .log
*.cache    # Tous les fichiers .cache
\`\`\`

### Dossiers spécifiques
\`\`\`
cache/     # Dossier cache complet
temp/      # Dossier temp complet
\`\`\`

### Fichiers système
\`\`\`
.DS_Store  # macOS
Thumbs.db  # Windows
\`\`\`

### Fichiers d'éditeur
\`\`\`
.vscode/   # VS Code
.idea/     # IntelliJ
*.swp      # Vim swap files
*.swo
\`\`\`

### Fichiers sensibles
\`\`\`
*.key      # Clés privées
*.pem      # Certificats
.env       # Variables d'environnement
\`\`\`

## Test de la configuration

\`\`\`bash
# Créer un fichier temporaire
echo \"test\" > notes.tmp

# Vérifier qu'il est ignoré
git status
# notes.tmp n'apparaît PAS dans git status

# Vérifier explicitement
git check-ignore -v notes.tmp
# Affiche: .gitignore:2:*.tmp	notes.tmp
\`\`\`

## Commandes utiles

| Commande | Description |
|----------|-------------|
| \`git status\` | Voir les fichiers ignorés (ne les affiche pas) |
| \`git check-ignore -v <fichier>\` | Vérifier si un fichier est ignoré et par quelle règle |
| \`git ls-files\` | Lister tous les fichiers trackés |
| \`git add -f <fichier>\` | Forcer l'ajout d'un fichier ignoré |

## Bonnes pratiques

1. **Versionner .gitignore** : Le fichier .gitignore doit être commité
2. **Ignorer tôt** : Configurer .gitignore avant d'ajouter des fichiers
3. **Ne pas ignorer des fichiers importants** : Vérifier qu'on n'ignore pas des fichiers nécessaires au projet
4. **Utiliser des patterns génériques** : \`*.tmp\` plutôt que \`notes.tmp\`
5. **Documenter** : Ajouter des commentaires dans .gitignore pour expliquer les règles

## Pour ignorer un fichier déjà tracké

Si un fichier est déjà versionné et qu'on veut l'ignorer :

\`\`\`bash
# Retirer du tracking sans supprimer le fichier
git rm --cached <fichier>

# Ajouter au .gitignore
echo \"<fichier>\" >> .gitignore

# Committer le changement
git add .gitignore
git commit -m \"chore: ignorer <fichier>\"
\`\`\`
