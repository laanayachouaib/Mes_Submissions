# Mon Projet Git

## Description
Ce projet est un exercice d\'apprentissage de Git et GitHub. Il contient une série de challenges pour maîtriser les concepts fondamentaux du version control.

## Challenges réalisés
- ✅ **C01** : Installation et setup - Initialisation du repo
- ✅ **C02** : Comprendre le version control - Documentation des concepts
- ✅ **C03** : Configuration Git - user.name et user.email
- ✅ **C04** : git status et staging - Ajout sélectif de fichiers
- 🔄 **C05** : Commit propre - Message descriptif (en cours)
